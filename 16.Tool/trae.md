#  快捷键

| 分类 | 功能 | macOS | Windows / Linux | 说明 |
|----|----|----|----|----|
| 核心导航 | 跳转到定义 | F12 / Cmd + 点击 | F12 / Ctrl + 点击 | 进入函数 / 变量定义 |
| 核心导航 | 回到上一个位置 | Ctrl + - | Alt + ← | 从定义回到调用处 |
| 核心导航 | 前进到下一个位置 | Ctrl + Shift + - | Alt + → | 撤销回退 |
| 代码理解 | 查看引用 | Shift + F12 | Shift + F12 | 查看函数 / 变量被调用位置 |
| 代码理解 | 速览定义（不跳转） | Option + F12 | Alt + F12 | 悬浮预览定义 |
| 代码理解 | 转到实现 | Cmd + F12 | Ctrl + F12 | 接口 → 实现 |
| 代码理解 | 重命名符号 | F2 | F2 | 统一修改变量 / 方法 |
| 文件导航 | 快速打开文件 | Cmd + P | Ctrl + P | 模糊搜索文件 |
| 文件导航 | 最近文件切换 | Ctrl + Tab | Ctrl + Tab | 在打开文件间切换 |
| 文件导航 | 文件内符号 | Cmd + Shift + O | Ctrl + Shift + O | 当前文件函数列表 |
| 文件导航 | 全局符号搜索 | Cmd + T | Ctrl + T | 全项目函数 / 类 |
| 光标定位 | 跳转到行号 | Ctrl + G | Ctrl + G | 快速定位指定行 |
| 光标定位 | 上一个光标位置 | Ctrl + - | Alt + ← | 光标历史回退 |
| 光标定位 | 下一个光标位置 | Ctrl + Shift + - | Alt + → | 光标历史前进 |
| 搜索 | 当前文件搜索 | Cmd + F | Ctrl + F | 文件内查找 |
| 搜索 | 全局搜索 | Cmd + Shift + F | Ctrl + Shift + F | 项目级搜索 |
| 代码折叠 | 折叠当前代码块 | Cmd + Option + [ | Ctrl + Shift + [ | 收起函数 / 代码块 |
| 代码折叠 | 展开当前代码块 | Cmd + Option + ] | Ctrl + Shift + ] | 展开函数 / 代码块 |
| 代码折叠 | 折叠全部 | Cmd + K Cmd + 0 | Ctrl + K Ctrl + 0 | 折叠整个文件 |
| 代码折叠 | 展开全部 | Cmd + K Cmd + J | Ctrl + K Ctrl + J | 展开整个文件 |
| 命令面板 | 打开命令面板 | Cmd + Shift + P | Ctrl + Shift + P | 所有功能入口 |
